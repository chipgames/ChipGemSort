# ChipGemSort 이름 및 잼 스타일 사용 검토

> ChipGemSort 프로젝트명 저작권·상표 이슈, ChipPuzzleGame 잼(젬) 모양 참고 가능성 정리

---

## 1. "ChipGemSort" 이름 사용

### 1.1 저작권·상표 관점

| 질문                                         | 답변                                                                                                                                                                                                                                                                              |
| -------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **ChipGemSort를 쓰면 저작권 문제가 되나요?** | **아니요.** 게임 이름·타이틀 자체는 저작권으로 보호되지 않으며, "ChipGemSort"는 Woody Sort·Sort Jam과 **다른 조합**의 자체 명칭입니다.                                                                                                                                            |
| **상표권 문제는?**                           | **"Sort Jam"**(Speedgo 앱)과는 **단어 순서·브랜드가 다릅니다.** Chip**GemSort** vs **Sort** Jam → 구분 가능한 별개 표지로 보는 것이 자연스럽습니다. 상표는 "동일·유사 상표로 동일·유사 상바에서 사용"할 때 문제되므로, Chip 브랜드 + GemSort 조합은 일반적으로 리스크가 낮습니다. |
| **실무 권장**                                | 앱/웹 서비스 출시 전에 **상표 검색**(특허청 키프리스 등)으로 "ChipGemSort"·"Chip Gem Sort" 유사 등록 여부만 한 번 확인하면 더 안전합니다.                                                                                                                                         |

**결론: ChipGemSort라는 이름을 사용하는 것만으로는 저작권·상표권 문제가 되지 않습니다.** (다만 상표 검색은 권장.)

## 2. ChipPuzzleGame의 "잼 모양" 참고

### 2.1 ChipPuzzleGame에서 확인된 내용

- ChipPuzzleGame에는 **"잼(jam)"**이라는 명시적 표현은 없고, **"젬(Gem)"**(보석) 오브젝트가 사용됩니다.
- **젬 시각 스타일**: `GemRenderer.ts` 기준
  - **모양**: 둥근 사각형(`roundRect`, radius 비율 약 0.2)
  - **색**: `gemConfig.ts`의 6색(red, yellow, blue, green, purple, orange) + light/dark 그라데이션
  - **패턴**: 색상별로 원·별·다이아몬드·삼각형·사각형·육각형 등 내부 패턴
  - **테두리·하이라이트**: 색상별 테두리, 상단 하이라이트

즉, 말씀하신 **"잼 모양"**은

- **ChipPuzzleGame의 젬처럼** 둥글고 그라데이션 있는 **블롭/드롭 형태**를 잼(jam) 테마로 쓰겠다는 의미로 이해했습니다.

### 2.2 ChipGemSort에서 그 스타일을 쓰는 경우

| 상황                                                    | 판단                                                                                                                                                                                                                                                                                                   |
| ------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **ChipPuzzleGame과 ChipGemSort가 같은 제작자(본인/팀)** | **문제 없음.** 본인이 만든 ChipPuzzleGame의 디자인·색감·형태를 ChipGemSort에서 "잼 느낌"으로 재해석해 쓰는 것은 **자기 저작물 재활용**이므로 저작권 문제가 되지 않습니다.                                                                                                                              |
| **ChipPuzzleGame 코드/에셋을 그대로 복사**              | ChipGemSort는 **다른 프로젝트**이므로, 코드·리소스를 가져다 쓸 때는 **라이선스·저작권**을 지켜야 합니다. ChipPuzzleGame이 MIT 등 오픈소스라면 라이선스 조건만 지키면 됩니다. **디자인 아이디어**(둥근 형태, 6색, 그라데이션)를 참고만 하고 **ChipGemSort용으로 새로 그리거나 구현**하면 더 명확합니다. |
| **"잼" 테마만 차용**                                    | "잼처럼 생긴 색 블롭"은 **아이디어에 가깝고**, 저작권은 **구체적 표현**(도형·색 코드·코드)을 보호합니다. 잼처럼 동글동글·반짝이는 느낌으로 **자체 디자인**하면 문제되지 않습니다.                                                                                                                      |

**결론: ChipPuzzleGame의 젬을 참고해 "잼 모양"으로 비슷한 스타일(둥근 형태, 그라데이션, 6색)을 ChipGemSort에 쓰는 것은, 같은 제작자라면 전혀 문제 없고, 다른 프로젝트라도 아이디어를 참고해 자체 제작하면 저작권상 문제될 것이 없습니다.**

## 3. 정리 체크리스트

- [ ] **이름**: ChipGemSort 사용 → 저작권·상표권상 무리 없음 (선택: 상표 검색)
- [ ] **잼 모양**: ChipPuzzleGame 젬 스타일 참고 → 같은 제작자면 OK, 코드/에셋은 복사하지 말고 ChipGemSort용으로 새로 구현
- [ ] **타사 게임**: Woody Sort, Sort Jam의 **이름·캐릭터·UI·아트**는 사용하지 않기

---

## 4. 요약 한 줄

**ChipGemSort라는 이름을 쓰는 것과, ChipPuzzleGame의 잼(젬) 모양을 참고해 ChipGemSort에서 비슷한 스타일로 자체 제작하는 것은 저작권·상표권상 문제될 것이 없습니다.**
