# 색상/워터 소트 퍼즐 게임 저작권 및 법적 분석

> Woody Sort, Sort Jam 등 동일 장르 게임 제작 시 저작권·법적 리스크 검토

---

## 1. 요약 결론

**동일한 장르(색상 분류·워터 소트 퍼즐)의 게임을 새로 만드는 것은 저작권법상 원칙적으로 문제되지 않습니다.**  
게임의 **아이디어, 규칙, 플레이 방법**은 저작권으로 보호되지 않으며, **자체 표현(UI, 캐릭터, 스토리, 아트, 음악 등)**만 독자적으로 만들면 됩니다.

---

## 2. 저작권이 보호하는 것 vs 보호하지 않는 것

### 2.1 보호되지 않는 것 (자유롭게 활용 가능한 요소)

| 항목                 | 설명                                                               |
| -------------------- | ------------------------------------------------------------------ |
| **게임 아이디어**    | “색을 병/튜브에 나눠 담아 정렬한다”는 아이디어 자체                |
| **게임 규칙·방법**   | 한 병에는 한 색만, 위에서부터 붓는다, 빈 병/같은 색 위에만 등 규칙 |
| **게임 메커니즘**    | 터치/클릭으로 병을 선택하고 액체/공을 옮기는 인터랙션 구조         |
| **게임 이름/타이틀** | 단순 제목은 저작권보다 상표권 이슈(타사 상표 사용만 피하면 됨)     |

미국 저작권청(U.S. Copyright Office) 지침에 따르면:

- “저작권은 **게임의 아이디어, 이름·타이틀, 또는 플레이하는 방법**을 보호하지 않는다.”
- “게임을 **개발·상품화·플레이하는 과정에 관련된 아이디어, 시스템, 방법, 장치**도 보호 대상이 아니다.”
- 저작권이 보호하는 것은 “**문학·예술·음악 형태로 표현된 구체적 표현**”이다.

따라서 **‘색/액체를 병에서 병으로 옮겨 정렬한다’는 메커니즘**은 저작권 대상이 아니며, 이 메커니즘을 사용한 새 게임을 만드는 것 자체는 저작권 침해가 되지 않습니다.

### 2.2 보호되는 것 (복제·모방 시 주의할 요소)

| 항목            | 설명                                             | 제작 시 유의사항                                               |
| --------------- | ------------------------------------------------ | -------------------------------------------------------------- |
| **시각적 표현** | 캐릭터, 병/튜브 디자인, 배경, 아이콘, 애니메이션 | 직접 복사하지 말고, 독자적인 디자인·스타일 사용                |
| **문구·텍스트** | 게임 내 문구, 튜토리얼, 스토리 텍스트            | 원문 그대로 쓰지 말고, 직접 작성 또는 완전히 다른 표현         |
| **음악·효과음** | BGM, SE                                          | 라이선스 있는 자체 제작/구매·무료 음원 사용                    |
| **코드·에셋**   | 소스코드, 리소스 파일                            | 타 게임 코드/에셋 사용 금지, 전부 직접 제작 또는 정식 라이선스 |

**Woody Sort, Sort Jam을 “참고”만 하고,**  
규칙·메커니즘은 같은 장르를 쓰되, **이름·캐릭터·UI·아트·음악·코드는 전부 본인(또는 팀)이 새로 만든다면** 저작권 측면에서는 문제될 여지가 없습니다.

---

## 3. 워터/컬러 소트 퍼즐 장르의 특성

- 이 장르는 **여러 회사·개발자가 비슷한 규칙으로 수많은 게임을 출시**한 “공개된 퍼즐 형태”에 가깝습니다.
- Poki만 봐도 **Water Color Sort, Fun Water Sorting, Emoji Sort Master, Market Sort** 등 동일한 “색/물체 정렬” 메커니즘의 게임이 다수 존재합니다.
- 학술적으로도 **컬러 소트 퍼즐은 NP-완전**으로 연구되며, “일반적인 퍼즐 타입”으로 취급됩니다.

즉, **특정 한 게임만의 독점적 아이디어**가 아니라 **공통된 퍼즐 장르**이므로, 이 장르를 선택해 새 게임을 만드는 것은 저작권상 허용되는 범위입니다.

---

## 4. 제작 시 체크리스트 (저작권·법적)

- [ ] **게임 규칙·메커니즘**: 같은 “색 정렬 퍼즐”이라도, 텍스트·규칙 설명은 본인이 작성했는지 확인
- [ ] **캐릭터·UI·아트**: Woody Sort / Sort Jam의 캐릭터·로고·병 디자인·배경을 그대로 쓰지 않았는지 확인
- [ ] **이름·로고**: “Woody Sort”, “Sort Jam” 등 타사 게임명·로고를 그대로 사용하지 않음 (상표권·부정경쟁 이슈 방지)
- [ ] **음악·SE**: 모든 오디오는 본인 제작, 또는 라이선스/무료 음원만 사용
- [ ] **코드·에셋**: 오픈소스/에셋 스토어 사용 시 각 라이선스(예: MIT, CC) 준수
