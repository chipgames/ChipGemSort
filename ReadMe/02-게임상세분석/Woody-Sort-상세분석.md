# Woody Sort 게임 상세 분석

> 출처: [Poki - Woody Sort (KR)](https://poki.com/kr/g/woody-sort)  
> 제작: **Unico Studio**

---

## 1. 개요

| 항목       | 내용                             |
| ---------- | -------------------------------- |
| **플랫폼** | 웹 (Poki), PC·모바일·태블릿 지원 |
| **장르**   | 퍼즐, 색상 매칭/분류, 두뇌 게임  |
| **조작**   | 마우스 (클릭/드래그로 공 이동)   |
| **가격**   | 무료 (광고 수익 모델 추정)       |

---

## 2. 핵심 규칙 및 목표

- **목표**: 모든 **색깔 공**을 분류하여, **각 튜브(병)에 한 가지 색의 공만** 들어 있게 만든다.
- **승리 조건**: 위 조건을 만족하면 해당 레벨 클리어 → 다음 레벨 진행.
- **제한**: 한 번에 옮길 수 있는 공의 개수·방향 등은 게임 내 규칙에 따름 (일반적으로 “같은 색이 쌓인 맨 위 덩어리”만 이동 가능한 형태일 가능성 높음).

---

## 3. 진행 구조

- **레벨 제도**: 스테이지별로 난이도가 올라가는 구조.
- **챌린지 3종**: 각 레벨마다 완료할 수 있는 **3가지 챌린지** 존재.
- **별 등급**: 3개 챌린지를 모두 완료하면 **3성** 획득.
- **힌트**: 막혀도 **힌트 요청**으로 다음 수를 안내받을 수 있음.

---

## 4. 플레이 방법 (조작)

- **마우스**로 공을 선택하고, 목표 튜브로 이동시켜 정렬.
- 터치 기기에서는 탭/드래그로 동일한 조작 가능 (Poki는 모바일 지원).

---

## 5. 분류 태그 (Poki 기준)

- 색상 게임
- 기억력 게임
- 공 게임
- 논리 게임
- 퍼즐 게임
- 마우스 게임
- 두뇌 게임

→ **타깃**: 캐주얼·두뇌 퍼즐을 즐기는 전 연령.

→ **같은 “색/객체 정렬” 메커니즘**을 쓰는 다른 테마·표현의 게임들이 이미 다수 존재.

---

## 7. 개발 시 참고 포인트

| 요소         | Woody Sort 특징      | 우리 게임에 적용 시                   |
| ------------ | -------------------- | ------------------------------------- |
| **오브젝트** | **공(ball)**         | 공 / 액체 / 블록 등으로 차별화 가능   |
| **진행**     | 레벨 + 3챌린지 + 3성 | 레벨/별/도전과제 구조 참고 가능       |
| **도움**     | 힌트                 | 힌트/언두/셔플 등 선택 구현           |
| **조작**     | 마우스 중심          | 마우스 + 터치 동시 지원 권장          |
| **비주얼**   | Unico 고유 스타일    | **독자 캐릭터·UI·테마**로 차별화 필수 |

---

## 8. 요약

- **Woody Sort**는 “**튜브에 색깔 공을 한 색씩 정렬**”하는 웹 기반 캐주얼 퍼즐이다.
- 레벨 + 3챌린지 + 3성 + 힌트 구조로 **재플레이·수집 욕구**를 부여한다.
- 메커니즘은 차용 가능하되, **이름·캐릭터·비주얼·사운드는 반드시 독자 제작**해야 저작권·상표권 리스크를 피할 수 있다.
