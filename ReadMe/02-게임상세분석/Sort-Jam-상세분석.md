## 1. 개요

| 항목       | 내용                                   |
| ---------- | -------------------------------------- |
| **플랫폼** | Android (Google Play)                  |

---

## 2. 컨셉 및 테마

- **“손가락 속의 마법”** – 색으로 가득한 세계에서 **액체 퍼즐**로 두뇌를 시험.
- **“물을 따르는 순간마다 탐험”** – 액체를 병에서 병으로 옮기는 행동이 핵심.
- 난이도가 **점점 어려워지는 레벨**과 **부드럽고 만족스러운 게임플레이** 강조.

---

## 3. 핵심 규칙 및 목표

- **목표**: 여러 색의 **액체(물)**를 병/튜브에 나눠 담아, **한 병에 한 색만** 있도록 정렬.
- **조작**: 병을 탭해 선택 → 다른 병으로 “따르기” (일반적인 워터 소트 규칙).
- **제한**: 보통 “맨 위 색만 붓기”, “빈 병 또는 같은 색 위에만 붓기” 등 규칙 적용.

---

## 4. 주요 기능

| 기능                | 설명                                             |
| ------------------- | ------------------------------------------------ |
| **실행 취소(Undo)** | 마지막 이동을 되돌림. 막혔을 때 유용.            |
| **셔플(Shuffle)**   | 병 순서/배치를 바꿔 새로 시작.                   |
| **부스트**          | “신비로운 부스트” 등으로 정렬을 도와주는 아이템. |
| **레벨 구성**       | 쉬운 난이도 ~ 전문가 난이도까지 다양한 퍼즐.     |
| **보상·업적**       | 레벨 진행 시 보상·업적 잠금 해제.                |

---

## 5. 비주얼·경험

- **선명한 색상** + **흐르는 물** 애니메이션으로 시각적 만족감 제공.
- **“클래식 워터 분류 퍼즐 + 마법 같은 요소”**로 차별화된 분위기.
- 빠른 캐주얼 세션과 긴 퍼즐 플레이 모두 타깃.

---

## 10. 요약

- **Sort Jam**은 **액체(물)를 병에서 병으로 옮겨 한 색씩 정렬**하는 모바일 캐주얼 퍼즐이다.
- 실행 취소·셔플·부스트로 **막힘을 줄이고** 보상·업적으로 **이탈 방지**를 꾀한다.
- 메커니즘은 장르 공통이므로 참고 가능하고, **이름·UI·비주얼·사운드는 독자 제작**해야 한다.
