### 5. HelpScreen 내용 보강

**현재 상태:**

- 기본 구조만 있고 FAQ 내용 없음

**필요 작업:**

- FAQ 항목 추가 (로케일 파일에 `help.faq` 섹션)
- 질문/답변 형식으로 구성
- 예: "게임이 너무 어려워요", "스테이지를 건너뛸 수 있나요?", "진행 상황이 저장되나요?" 등

---

### 6. 가이드 화면 개선

**현재 상태:**

- 기본 텍스트만 있음

**필요 작업:**

- 시각적 예시 추가 (Canvas로 간단한 애니메이션 또는 이미지)
- 단계별 설명 보강
- 게임 규칙 상세 설명

### 7. AboutScreen 내용 보강

**현재 상태:**

- 기본 설명만 있음

**필요 작업:**

- 게임 특징, 제작자 정보, 크레딧 등 추가
- 로케일 파일에 `about` 섹션 확장
